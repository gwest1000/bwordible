# bWORDibLE

bWORDibLE is a Bible-themed daily word puzzle with 4 to 6 letter answers.

## Local development

```bash
npm install
npm start
```

Open `http://127.0.0.1:4173`.

## Tests

```bash
npm run test:e2e
```

## GitHub Pages

This repo includes a GitHub Actions workflow that deploys the static app to GitHub Pages on every push to `main`.
